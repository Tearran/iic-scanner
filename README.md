# Inter-Integrated Circuit - Scanner

## iic-scanner
An alternative to  i2cdetect

download an copy to  /usr/bin/
enable i2c for yor device
## Features :
- [x] dispaly i2c (csv)
- [x] dispaly useage
- [ ] user options
- [ ] enable disable i2c
  
## useage :
- iic-scanner <device #> 
```bash
iic-scanner 1
```
